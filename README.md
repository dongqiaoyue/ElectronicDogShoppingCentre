# BaseAdmin

#### 项目介绍
yii2 做的基础的开箱即用的 RBAC 基础后台。可用于各种中后台系统开发

#### 软件架构
软件架构说明


#### 安装教程

1. git clone https://gitee.com/nickbai/BaseAdmin.git
2. composer install
3. init
3. config 中配置数据库连接
4. 导入根目录下的 system.sql

#### 使用说明

1. 配置 虚拟域名指向 backend
2. 访问虚拟域名 进入 管理后台，默认账号密码： admin  admin
3. php 版本>=5.5
